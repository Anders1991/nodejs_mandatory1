Javascript Parsing

You can change different types of data to other types using parsing. An example of this is the following:

// Exercise 1 - Add numbers from string to float
const numberOne = "1.10";
const numberTwo = "2.30";
// add those two numbers and show the result
// you cannot touch line 1 neither line 2
console.log(parseFloat(numberOne) + parseFloat(numberTwo));

Here the two constants numberOne and numberTwo are strings, that become parsed using parseFloat() as floats, allowing them to be added together.

Another example of this is using the json.parse function in node to turn strings into json and json.stringify to turn json into strings.
