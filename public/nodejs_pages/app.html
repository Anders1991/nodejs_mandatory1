nodejs app.js

What is it?

This file is where we define all of our server routing, which port the server should listen to and what it should do with certain calls.

Setting up

First we need to import the library express as follows:

const { json } = require("express");
const express = require("express");

We then make an instantiated instance of express

app.use(express.json());

Now we can start using express to handle http requests for our server. As such lets take a look at the following code:

app.get("/", (req, res) => {
res.sendFile('index.html',{root:'public'});
});

Here we ask express to handle a GET request for our server, by responding with sending the file index.html to the user.

We could also require some data from the user, and send something appropriate back.

Last but not least we should allow the server to run on a port:

app.listen(8080);

Here we ask express to listen to the port 8080 (localhost) for our app.
