Rest API

What is it?

A REST API (Representational State Transfer Application Programming Interface) is a data collection accessible by HTTP requests. As an example, to get the information from a JSON Rest API you would have to send a GET HTTP request to the server that has the data.

CRUD

the CRUD operations are Create, Read, Update, Delete. To use these operations on an URL endpoint in nodejs do the following:

When you want to Create new data use a POST request, to post new data.

When you want to Read data from the API use a GET request, to retrieve data.

When you want to Update data in the API use a PUT request, to update the data.

When you want to Delete data from the API use a DELETE request, to remove the data.

Nodejs example

app.post("/name-of-the-endpoint", (req, res) => {

//put your code here

res.send();
