Nodemon

What is it?

Nodemon is a node module that is useful while developing a nodejs server. Nodemon automatically restarts the server whenever you make changes to the code, allowing you to see the changes.

Setting up

To use Nodemon you must first install it. Simply run the following terminal command:

npm install nodemon

Head to your package.json file and add the following code:

"scripts": {
"dev": "nodemon app.js"
}

Using Nodemon

to start Nodemon use the following terminal command:

npm run dev

This lets Nodemon know it should run the app.js file (because we declared it above).
