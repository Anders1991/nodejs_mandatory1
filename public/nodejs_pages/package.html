nodejs Package.json

What is it?

The package.json file is where we define our dependencies such as the external library Express. We also define the version of nodejs we want.

We also define scripts we can run from the terminal. When hosting on Heroku it is sometimes necessary to include the following script to run the server:

"scripts": { "start": "node app.js" }

The only thing it does is run the file app.js (which is very important!).
